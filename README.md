# Tank Modern - Glassmorphism Edition

Chào mừng bạn đến với Tank Modern! Đây là một trò chơi bắn xe tăng hiện đại với giao diện cực đẹp theo phong cách Glassmorphism.

## Hướng dẫn chơi
1. **Bắt đầu**: Nhấn nút "BẮT ĐẦU" ở màn hình chính.
2. **Di chuyển**: Sử dụng các phím **WASD** hoặc các phím **Mũi tên**.
3. **Ngắm bắn**: Di chuyển chuột để xoay nòng súng.
4. **Bắn**: Nhấn chuột trái hoặc phím **Space** để bắn đạn.
5. **Mục tiêu**: Tiêu diệt càng nhiều xe tăng địch (màu đỏ) càng tốt và giữ cho lượng máu (Health) không về 0%.

## Điểm nhấn thẩm mỹ
- **Glassmorphism**: Các bảng thông báo và HUD có hiệu ứng nền mờ bán trong suốt.
- **Gradients & Shadows**: Sử dụng bảng màu xanh dương sâu và hiệu ứng đổ bóng hiện đại.
- **Micro-animations**: Hiệu ứng rung động ở nút Start và hiệu ứng hạt (particles) khi có vụ nổ.

## Cách chạy dự án
Chỉ cần mở file `index.html` trong trình duyệt web của bạn hoặc chạy lệnh sau trong terminal:
```bash
npx serve .
```
Hoặc đơn giản là mở bằng Live Server trong VS Code.
